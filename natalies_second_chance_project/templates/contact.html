Surrendering a dog

If you are in need of surrendering a dog please send us a detailed description of the dog and a picture if possible, We will try our best to get back with you quickly. We cannot guarantee that we will have an available place. We have limited kennel space so we cannot take in every dog but we can courtesy post to try and find them a home.

Found a dog

If you have found a dog please send us a picture and exact location found we will be able to post them and hopefully get them home! We are also able to scan found dogs for microchips containing owner information.
